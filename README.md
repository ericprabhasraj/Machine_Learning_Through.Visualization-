# Machine_Learning_Through.Visualization-

This project is a fully interactive Machine Learning learning platform designed to simplify complex concepts through visualization and hands-on exploration—without requiring any external libraries.

It includes:

A Decision Boundary Visualizer supporting KNN, Logistic Regression, and SVM with live accuracy updates
Probability & Bayes module with coin flip simulations, Gaussian distribution controls, and a Bayes theorem calculator
Neural Network simulator with adjustable layers and activation functions (ReLU, Sigmoid, Tanh)
Gradient Descent visualizer showing optimization behavior across different loss landscapes

Built for students, developers, and anyone looking to develop an intuitive understanding of core ML concepts.
